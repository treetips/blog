<a id="システム面"></a>

### システム面

<a id="ダッシュ機能"></a>

#### ダッシュ機能

FC版には無いダッシュ機能がPSP版にはあります。通常の2倍の速度あるけるようになる機能で、これが非常に快適です。 しかしこのダッシュ機能のせいか解りませんが、エンカウントが低いように感じました。中々敵と遭遇しません。おかげでダンジョンで苦戦した箇所は1箇所もありませんでした。

<a id="どこでも中断"></a>

#### どこでも中断

DS版でも同じだったのか解りませんが、ゲーム中にどんな状態であれ、電源ボタンを押してスリープ状態にする事で、ゲームの時間を止めつつどこでも中断できてしまいます。これは便利ですが、なんかビギナー向けに難易度が下がった感じがしてちょっと残念。

<a id="BGMをFC版とアレンジ版に切り替え可能"></a>

#### BGMをFC版とアレンジ版に切り替え可能

FC版のBGMが懐かしすぎて泣けてきました。たまにFC版とかに切り替えると気分転換になります。

<a id="いい感じのアレンジ曲が多い"></a>

#### いい感じのアレンジ曲が多い

クリスタルタワー、闇の世界、水の巫女エリア、が特に素晴らしいアレンジでした！！ 特にクリスタルタワーはオーケストラ風で一瞬！？とビックリしました。

<a id="難易度"></a>

#### 難易度

難易度は非常に低いです。簡単過ぎます。雑魚敵はドラゴン系以外は非常に弱いです。 ボスは序盤こそターン数が必要ですが、中盤移行は基本2ターンキルです。 アーリマンや2ヘッドドラゴンですら2ターンキルという有り様で、難易度はかなり不満が残ります。

ザコ敵の中にドラゴン系が現れる事がありますが、これが強い。非常に強い。 他の雑魚のHPが1万前後の中、奴らはHPが7〜10万程度あり、クリスタル甲・鎧・腕で固めても1撃で即死する程の威力。たまにクリスタルタワーで出くわすので、注意。特に風属性に耐性があるので、竜騎士のダメージが半減します。

隠しボスの鉄巨人については暗闇の雲の10倍くらい強いでしょうか。 暗闇の雲を撃破した直後に挑んだところ、LV62のパーティーが2ターンキルされました。。

<a id="限界突破"></a>

#### 限界突破

今作は画面上は9999ダメージが最高に見えますが、実は違います。 表示上が9999なだけで、実際は9999という限界を突破したダメージを与えています。 インビジブルで最大ダメージを確認できますが、私の場合は39000でした。 多分序盤の飛行型に竜騎士の風特効でジャンプした時？かもしれません。

後半は簡単に限界突破するので、FC版と比べて難易度が下がる大きな要因となっていると思われます。 狩人の乱れ打ちや戦士のふみこみなんて一体どれ程のダメージになっているか解らない程当たってます。

<a id="ジョブ"></a>

### ジョブ

最終的なパーティージョブは、狩人・竜騎士・吟遊詩人・導師、でした。 魔剣士や忍者も気分転換で入れたりもしてました。 今回は私が実際に使ったジョブの中で簡単にランキングしたいと思います。

<a id="役に立つランキング"></a>

#### 役に立つランキング

   1位 戦士 多分最強キャラです。素早さは速めでふみこむのダメージが大きすぎ。序盤から使えるので成長も楽。強いですがつまらないので今回使いませんでした。熟練度99にしましたが。   2位 狩人 みだれうちが強い。最終的に1回で2〜3万程度のダメージになります。更に4回行動するので熟練度と腕熟練度が上がり易く、成長が速いです。後列が基本なので被ダメは少ない。   3位 吟遊詩人 うたうは2ターン持続・必ず最速で始動・必中・全員に適用、という特徴を持ちます。まもりの歌で被ダメを減らし、破滅の歌でボスキャラに割合ダメージ、という有能なキャラ。LV60だと癒しの歌で600以上回復。   4位 導師 唯一まともな回復専門キャラ。ケアルダで2000以上回復するのでケアルがの出番はほとんど無し。やる事がなく防御したり杖ふる事が多いです。   5位 竜騎士 実は強かったです。素早さが速く、常に最高の防具を装備でき、登場する槍がほとんど強い。グングニル＋まじんのやりでステータスupもおいしい。風特攻の武器も多く、飛行系モンスターで余裕のカンスト連発。ブラッドランスはブラッドソードより攻撃力がかなり高いので、古代の民の迷宮くらいまで活躍します。   6位 魔剣士 隠れた実力者。中盤装備できる防具に乏しく苦戦する期間がありますが、後半が強い。源氏・クリスタル装備可能で、ラグナロク・マサムネ・あまのむらくも等のステup系武器が装備可能。中盤〜後半までブラッドソード＋きくいちもんじ装備が大活躍。忍者と比較されますが、忍者はまもりの指輪とリボンを入手するまで防御が紙で苦労します。

2chでのオススメは、戦士・狩人・導師・赤魔道士、がよく挙がってました。 最強は戦士でほぼ結論が出ていて、次点で狩人・シーフ・赤魔道士、のようでした。

<a id="役に立たないランキング"></a>

#### 役に立たないランキング

   1位 魔人 文句なしの最弱。今作では魔法が弱すぎます。後半は物理が非常に強く、簡単にカンストするため、攻撃魔法の出番は一切ありません。今回風水師がそこそこ強力なので完全に出番無し。   2位 黒魔道士 同上。序盤は黒魔道士の魔法より白魔道士・赤魔道士が杖ふった方が強いのです・・   3位 幻術士 白召喚で回復やステアップがほんの少しだけ使えるかも？？しれません。が、弱い・・

2chでは、黒魔道士・魔人・幻術士・魔界幻士が文句無しのワーストという事で結論が出てました。。。 魔界幻士は唯一白バハムルのオーラが有能です。それ以外はゴミです。。。

<a id="微妙なジョブランキング"></a>

#### 微妙なジョブランキング

   1位 風水師 中盤は強いです。物理系ジョブ以上のダメージを全体化したようなダメージで、ハイン線で大活躍。後半が問題です。後半は物理系キャラが強いので、普通に物理で殴る方が強く、活躍期間が短い不遇なジョブ。シャドウフレアを引けばカンストしますが、中々でない。   2位 バイキング 完全に鉄巨人戦のためのキャラです。挑発＋両手盾＋後列で鉄壁の防御で敵の単体攻撃を完全に無力化できる性能を持ちますが、素早さが遅いが故成長が遅く、攻撃力は戦士系キャラより低いです。挑発は全体攻撃には無力という点もマイナス。   3位 赤魔道士 2chでの評価通り、中々のジョブです。特に序盤〜中盤は強力で、後列で重装備＋両手盾で被ダメが圧倒的に少なく、白魔道士が不要になります。しかし問題は後半。中途半端な攻撃力、中途半端な魔法使用回数、中途半端な魔力なのです。後半は完全に吟遊詩人に取って代わられます。   4位 ナイト 素早さが遅く、ヒット数が少ないので攻撃力は高くないです。防御は竜騎士や魔剣士と変わらず。自動かばう以外に魅力が無いジョブです。   5位 賢者 導師・魔人より魔法使用回数が少ないのでパーティーのLVが上がるまでは戦力として微妙。大体LV60以上くらいになればそもそも魔法使用回数が減るため、導師よりも活躍の場が増えます。

<a id="ボスダイジェスト"></a>

### ボスダイジェスト

<a id="ハイン"></a>

#### ハイン

風水師のちけい、狩人の乱れ打ち、赤魔道士の攻撃で2ターンキル。 風水師のちけいダメージが異次元過ぎ。

<a id="ガルーダ"></a>

#### ガルーダ

両手槍の竜騎士のジャンプ×2で終了。 逆にいかづちを食らうと1〜2ターンキルで終了。 潔い勝負でした。

<a id="オーディーンリバイアサンバハムート"></a>

#### オーディーン・リバイアサン・バハムート

吟遊詩人の攻撃の歌 → 狩人乱れ打ち → 竜騎士の攻撃、で2ターンキル。

<a id="ザンデ"></a>

#### ザンデ

吟遊詩人の攻撃の歌 → 狩人乱れ打ち → 竜騎士の攻撃、で2ターンキル。 ファファファ・・・しねい！はこっちのセリフだった。

<a id="ケルベロスエキドナ2ヘッドドラゴンアーリマン"></a>

#### ケルベロス・エキドナ・2ヘッドドラゴン・アーリマン

吟遊詩人の攻撃の歌 → 狩人乱れ打ち → 竜騎士の攻撃、で2ターンキル。 弱すぎて吹きました。。

<a id="暗闇の雲"></a>

#### 暗闇の雲

苦戦しました。何が苦戦したって？

狩人の乱れ打ちが散る

それだけです。上側の触手が物理無効なので、そっちに乱れ打ちが飛ぶのが面倒でした。 え？他に何かあったろって？

波動砲が1200しか喰らわないので暇だった

触手の臭い息は全員リボン装備で無効化。暗闇の雲の物理高原は500程度、そのうえ波動砲が1200なので、かなり暇でした。暇すぎて導師がホーリーうったり竜騎士がホーリーランスでホーリー撃ってました。

<a id="雑感"></a>

### 雑感

ちょっと難易度が低かったですね。多分3倍くらい難しくてもいいです。 グラフィックはいい感じの3D化でBGMもアレンジ曲は凄く良かったので、後は難易度ですね。 私としてはロマサガやメガテンのような雑魚敵で一瞬にして即死→全面の最凶パターンがあった方がハラハラして面白いですね。これもライトユーザーを取り込もうとする時代の流れなのでしょうか。。
