<a id="気になっている購入予定の本"></a>

### 気になっている・購入予定の本

[Gradle徹底入門 次世代ビルドツールによる自動化基盤の構築](http://www.amazon.co.jp/exec/obidos/ASIN/4798136433/treeapps5-22/) ![Gradle徹底入門 次世代ビルドツールによる自動化基盤の構築](https://images-fe.ssl-images-amazon.com/images/I/51EV2ZpCwLL._SL160_.jpg)これは絶対欲しい！！ 今一番欲しい書籍です。大型本ですよ！！100%買います！！ 仕事で使っていますが、使いこなせてない感が半端なく、公式リファレンスが微妙に解りづらい部分があったので、絶対買います。発売は2014/11/05とのこと！

[入門Backbone.js \(Programmer's SELECTION\)](http://www.amazon.co.jp/exec/obidos/ASIN/479813712X/treeapps5-22/) ![入門Backbone.js \(Programmer's SELECTION\)](https://images-fe.ssl-images-amazon.com/images/I/51tPECMzHIL._SL160_.jpg)angular.jsの日本の書籍も発売されましたが、個人的にangular.jsはある一定の段階を超えると急激に難易度が上がり、覚える事が多すぎるのが嫌な感じがします。なのでbackbone.jsが気になっています。個人的にはknockout.jsが滅茶苦茶解りやすく、旧ブラウザの対応もされてるので、knockout.jsの書籍が欲しいところです。

[現場のプロが教えるWebデザイン＆レイアウトの最新常識 知らないと困るWebデザインの新ルール3](http://www.amazon.co.jp/exec/obidos/ASIN/B00NXMFCWS/treeapps5-22/) ![現場のプロが教えるWebデザイン＆レイアウトの最新常識 知らないと困るWebデザインの新ルール3](https://images-fe.ssl-images-amazon.com/images/I/519x7-IWsQL._SL160_.jpg)tree-tipsやtree-maps等、新規に開発する時にbootstrapを使うのですが、どうレイアウトしたらいいか実は解らず、いつも適当にデザインしています。この辺のUI周りが解消できるかな。

[現場のプロが教えるWebディレクションの最新常識 知らないと困るWebデザインの新ルール2](http://www.amazon.co.jp/exec/obidos/ASIN/B00MVQY688/treeapps5-22/) ![現場のプロが教えるWebディレクションの最新常識 知らないと困るWebデザインの新ルール2](https://images-fe.ssl-images-amazon.com/images/I/51fW0U4mA9L._SL160_.jpg)会社でディレクションする人がいるのですが、ディレクションする人達は皆忙しく、ディレクションできてません。だから自分がやってしまおうか、と考えています。

[現場のプロがやさしく書いたWebサイトの分析・改善の教科書 Googleアナリティクスと、その他ツールを使った実践的ノウハウ](http://www.amazon.co.jp/exec/obidos/ASIN/B00MN07M1K/treeapps5-22/) ![現場のプロがやさしく書いたWebサイトの分析・改善の教科書 Googleアナリティクスと、その他ツールを使った実践的ノウハウ](https://images-fe.ssl-images-amazon.com/images/I/51sKjOxdyZL._SL160_.jpg)これはちょっと欲しいです。 正直いって分析系は自分は全くできません。 GAの使い方も解らなければ、サイトのどこを改善したらよいか解らない。それが解消できるかな。

<a id="購入済みの本"></a>

### 購入済みの本

[Web制作者のためのCSS設計の教科書 モダンWeb開発に欠かせない「修正しやすいCSS」の設計手法](http://www.amazon.co.jp/exec/obidos/ASIN/4844336355/treeapps5-22/) ![Web制作者のためのCSS設計の教科書 モダンWeb開発に欠かせない「修正しやすいCSS」の設計手法](https://images-fe.ssl-images-amazon.com/images/I/51rsbQOrZ0L._SL160_.jpg)kindle版を購入済みです。 この本は個人的にサイトを開発するとき、cssのセレクタがクソ設計になりがちなので購入しました。こちらはサイバーエージェントの方が執筆されています。

[開発効率をUPする Git逆引き入門](http://www.amazon.co.jp/exec/obidos/ASIN/4863541465/treeapps5-22/) ![開発効率をUPする Git逆引き入門](https://images-fe.ssl-images-amazon.com/images/I/51RtoUsheVL._SL160_.jpg)紙媒体を購入済みです。 この本のいいところはSourceTreeの解説なのですが、winとmacでメニューの出方等が実は違うのですが、mac使いの私がwin使いの方にSourceTreeの使い方を教える時はこの本を見せます。こちらもサイバーエージェントの方が執筆されています。

[入門Ansible](http://www.amazon.co.jp/exec/obidos/ASIN/B00MALTGDY/treeapps5-22/) ![入門Ansible](https://images-fe.ssl-images-amazon.com/images/I/51Ef9h7qfpL._SL160_.jpg)kindle版を購入済みです。 これ読めば大体ansibleの事を把握できます。私は小難しい使い方をしていないので、これだけ読めば知りたい事の9割は知る事ができました。

[高速スケーラブル検索エンジン ElasticSearch Server \(アスキー書籍\)](http://www.amazon.co.jp/exec/obidos/ASIN/B00J4KDYZU/treeapps5-22/) ![高速スケーラブル検索エンジン ElasticSearch Server \(アスキー書籍\)](https://images-fe.ssl-images-amazon.com/images/I/510nupY-9zL._SL160_.jpg)紙媒体を購入済みです。 Apache Solr 入門の執筆をされた大谷純さんが書かれたelasticsearch本です。 私は仕事でsolrを使っているのですが、そろそろelasticsearchを現場に投入するべく購入しました。
