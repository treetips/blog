先日ふとPS VITAを起動した時、以前作ったwifiの設定が効かなくなったようで、wifiに接続できなくなっていました。

<a id="症状"></a>

### 症状

無線LANはAppleのAirMac Extremeです。 wifiの設定自体は2年前くらいに行いました。その時からAirMac Extremeを使っていました。 設定自体はあっています。

<a id="対応"></a>

### 対応

根本的な解決ではありませんが・・・

無線LAN側でSSIDを変更し、VITA側でそれを指定したら繋がりました

結局根本的な解決策は解りませんでした。 キャッシュでも残っていたのでしょうかね。

<a id="最終的な設定"></a>

### 最終的な設定

設定の仕方も兼ねて、現在私の環境でこれで動いている、という設定の解説です。

<a id="無線LAN側の設定AirMac-Extreme"></a>

#### 無線LAN側の設定（AirMac Extreme）

![f:id:treeapps:20140628162524p:plain](/hatena-images/images/fotolife/t/treeapps/20140628/20140628162524.png) 暗号方式は「WPA／WPA2パーソナル」です。 VITAはこれより強い暗号方式（エンタープライズ等）に対応していない可能性があるので、WPA／WPA2パーソナルにしておきましょう。

<a id="VITA側の設定"></a>

#### VITA側の設定

![f:id:treeapps:20140628193658p:plain](/hatena-images/images/fotolife/t/treeapps/20140628/20140628193658.png) 前述で作成した新たなSSIDをVITAが見つけ出してくれるので、それを選択し、無線LAN側で設定したパスワードを入力するだけで、wifiに接続可能になります。
