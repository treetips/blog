[Bootstrap](http://twitter.github.com/bootstrap/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://twitter.github.com/bootstrap/\)](http://b.hatena.ne.jp/entry/http://twitter.github.com/bootstrap/) 最近急速に普及されたBootstrapですが、 テーマや[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)も揃ってきたので、現時点で存在するものをまとめてみました。

bootstrapテーマテーマを導入すると、UIを簡単に綺麗なものに変更する事ができます。

TODC Bootstrap[TODC Bootstrap](http://todc.github.com/todc-bootstrap/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://todc.github.com/todc-bootstrap/\)](http://b.hatena.ne.jp/entry/http://todc.github.com/todc-bootstrap/) [Google](http://d.hatena.ne.jp/keyword/Google)をリスペクトし、現在の[google](http://d.hatena.ne.jp/keyword/google)のUIをBootstrapで再現したテーマです。 最近の[google](http://d.hatena.ne.jp/keyword/google)のUIはフラットです。 ボタン等は[google](http://d.hatena.ne.jp/keyword/google)のボタンそのものです。再現度はかなり高いと思われます。 ![f:id:treeapps:20130220020439p:plain](/hatena-images/images/fotolife/t/treeapps/20130220/20130220020439.png)

Fbootstrapp[Fbootstrapp by Clemens Krack, based on Bootstrap, from Twitter](http://ckrack.github.com/fbootstrapp/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://ckrack.github.com/fbootstrapp/\)](http://b.hatena.ne.jp/entry/http://ckrack.github.com/fbootstrapp/) [Facebook](http://d.hatena.ne.jp/keyword/Facebook)調のUIをBootstrapで再現したテーマです。 [Facebook](http://d.hatena.ne.jp/keyword/Facebook)の特徴である、淡い青ベースの色調のテーマとなっています。 ![f:id:treeapps:20130220020457p:plain](/hatena-images/images/fotolife/t/treeapps/20130220/20130220020457.png)

BootMetro[Index](http://aozora.github.com/bootmetro/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://aozora.github.com/bootmetro/\)](http://b.hatena.ne.jp/entry/http://aozora.github.com/bootmetro/) [Windows8](http://d.hatena.ne.jp/keyword/Windows8)のメトロ調UIをBootstrapで再現したテーマです。 Metroの特徴であるフラットなUIです。 ![f:id:treeapps:20130220020514p:plain](/hatena-images/images/fotolife/t/treeapps/20130220/20130220020514.png)

BootstrapWP[509 Bandwidth Limit Exceeded](http://bootstrapwp.rachelbaker.me/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://bootstrapwp.rachelbaker.me/\)](http://b.hatena.ne.jp/entry/http://bootstrapwp.rachelbaker.me/) [WordPress](http://d.hatena.ne.jp/keyword/WordPress)にBootstrapのテーマを適用する事ができます。

Themes for [Twitter](http://d.hatena.ne.jp/keyword/Twitter) Bootstrap[WrapBootstrap - Bootstrap Themes & Templates](https://wrapbootstrap.com/)[!\[\]\(http://b.hatena.ne.jp/entry/image/https://wrapbootstrap.com/\)](http://b.hatena.ne.jp/entry/https://wrapbootstrap.com/) Bootstrapを使ったデザイン一式を配布しています。 ただし有償です。 Bootstrap自体はデザインし易いようにできているので、自力で作れる事も可能ですが、 面倒な方は購入してみてもいいかと思います。$15〜$20程度で購入できるものが多いようです。

bootstrap[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)拡張[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)を導入すると、Bootstrap内蔵の[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)を拡張又は追加する事ができます。

bootstrap-wysihtml5[bootstrap-wysihtml5](http://jhollingworth.github.com/bootstrap-wysihtml5/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://jhollingworth.github.com/bootstrap-wysihtml5/\)](http://b.hatena.ne.jp/entry/http://jhollingworth.github.com/bootstrap-wysihtml5/) Bootstrapに[wysiwyg](http://d.hatena.ne.jp/keyword/wysiwyg)エディタを追加する[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)です。 機能的に少ないので、シンプルな使い方をする場合は使えそうです。 ![f:id:treeapps:20130220020532p:plain](/hatena-images/images/fotolife/t/treeapps/20130220/20130220020532.png)

fuel UX[Fuel UX](http://exacttarget.github.com/fuelux/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://exacttarget.github.com/fuelux/\)](http://b.hatena.ne.jp/entry/http://exacttarget.github.com/fuelux/) Bootstrapに以下の拡張[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)を追加する事ができます。 沢山の機能があるのでおすすめです。 Wizard（パンくず）は見た目も綺麗で良い感じです。

- Checkbox
- Combobox
- Datagrid
- Pillbox
- Radio
- Search
- Select
- Spinner
- Tree
- Wizard

Bootstrap Arrows.[Bootstrap Arrows](http://bootstrap-arrows.iarfhlaith.com/index.html)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://bootstrap-arrows.iarfhlaith.com/index.html\)](http://b.hatena.ne.jp/entry/http://bootstrap-arrows.iarfhlaith.com/index.html) Bootstrapに「矢印のみを追加する」という潔い[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)です。 大小7色の矢印が用意されています。 ![f:id:treeapps:20130220020620p:plain](/hatena-images/images/fotolife/t/treeapps/20130220/20130220020620.png)

Backgrid.js[Backgrid.js - A powerful widget set for building data grids with Backbone.js](http://wyuenho.github.com/backgrid/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://wyuenho.github.com/backgrid/\)](http://b.hatena.ne.jp/entry/http://wyuenho.github.com/backgrid/) Bootstrapのグリッド（テーブル）を拡張する[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)です。 グリッドに対してカラム毎に昇順・降順ソート可能、ページングをサポート、その他細かいカスタマイズも可能です。 ![f:id:treeapps:20130220020636p:plain](/hatena-images/images/fotolife/t/treeapps/20130220/20130220020636.png)

[jQuery](http://d.hatena.ne.jp/keyword/jQuery) File Upload[jQuery File Upload Demo](http://blueimp.github.com/jQuery-File-Upload/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://blueimp.github.com/jQuery-File-Upload/\)](http://b.hatena.ne.jp/entry/http://blueimp.github.com/jQuery-File-Upload/) Bootstrapにファイルアップロードする[コンポーネント](http://d.hatena.ne.jp/keyword/%A5%B3%A5%F3%A5%DD%A1%BC%A5%CD%A5%F3%A5%C8)を追加する事ができます。 ファイルアップロード機能は非常にありがたいですね。 複数のファイルを同時に選択・アップロードする事ができます。 Bootstrapの[プログレスバー](http://d.hatena.ne.jp/keyword/%A5%D7%A5%ED%A5%B0%A5%EC%A5%B9%A5%D0%A1%BC)を利用して進捗状況もリアルタイムに反映されます。 ![f:id:treeapps:20130220020653p:plain](/hatena-images/images/fotolife/t/treeapps/20130220/20130220020653.png)
