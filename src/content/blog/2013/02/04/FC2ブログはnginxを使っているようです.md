偶然見てしまったのですが、 今現在FC2ブログは緊急メンテナンスをしているそうです。 で、何度かリロードしていたら、以下のメッセージが見えてしまいました。

```
500 Internal Server Error

nginx/1.1.19
```

ほほぅ、FC2ブログはnginx使ってたのか FC2ブログは[ライブドア](http://d.hatena.ne.jp/keyword/%A5%E9%A5%A4%A5%D6%A5%C9%A5%A2)やはてなと違って技術系な会社に見えないので、 てっきり安直に[apache](http://d.hatena.ne.jp/keyword/apache)使ってるのかと思ってました。 バージョンも新しいし、FC2も結構がんばってるんですね。
