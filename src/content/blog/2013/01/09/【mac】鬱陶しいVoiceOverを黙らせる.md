勝手に喋りだすの、結構ウザいんですよね。

![f:id:treeapps:20170829002500p:plain](/hatena-images/images/fotolife/t/treeapps/20170829/20170829002500.png)

Firefox等のブラウザを使っている場合、Ctrl + F5でキャッシュ無効化でリロードする事ができます。

しかしmacでそれをやってしまうと・・・VoiceOverが起動するのです。

これがもう・・・

鬱陶しい！鬱陶しい！鬱陶しい！鬱陶しい！

もうね、勝手にベラベラ喋り出すので殺意が沸くのです。

誤って起動してしまうが、オフにする方法が解らない！！一体どこでオフにするんだ！？次から次へとベラベラ喋って鬱陶しいしイライラする！こんな経験無いでしょうか。

VoiceOverの声でイライラしないためにもこいつを黙らせましょう。

- [ON/OFFのショートカットキー](#ONOFFのショートカットキー)
- [環境設定からOFFにする](#環境設定からOFFにする)
- [誤って起動しないよう抹殺する](#誤って起動しないよう抹殺する)

<a id="ONOFFのショートカットキー"></a>

### ON/OFFのショートカットキー

Command + F5でON/OFFを切り替え可能です。

<a id="環境設定からOFFにする"></a>

### 環境設定からOFFにする

環境設定 → アクセシビリティ → VoiceOver → 「VoiceOverを有効にする」をオフにする

![f:id:treeapps:20130109220335p:plain](/hatena-images/images/fotolife/t/treeapps/20130109/20130109220335.png)

<a id="誤って起動しないよう抹殺する"></a>

### 誤って起動しないよう抹殺する

諸悪の根源はショートカットキーが割り当てられているのが原因です。

なので、ショートカットキーを解除してしまうことで封殺します。

環境設定 → キーボード → アクセシビリティ → 「VoiceOverの入／切」をオフ

![f:id:treeapps:20130109220733p:plain](/hatena-images/images/fotolife/t/treeapps/20130109/20130109220733.png)

これで鬱陶しいVoiceOverともお別れできましたね！！

ざまーみろVoiceOverめ！！

これで安心してエロ動画鑑賞に打ち込めるぜ！
