動きますよ〜

![f:id:treeapps:20170829002500p:plain](/hatena-images/images/fotolife/t/treeapps/20170829/20170829002500.png)

Mountain LionからMavericksにアップデートしたのですが、brew updateしたらエラーが発生しました。 エラー発生から解決まで、まとめてみます。

最初にXCodeの5をインストールしました。 その状態でbrew updateすると、以下のようなエラーが発生しました。

```sh
treemacpro:project tree$ brew update
error: Your local changes to the following files would be overwritten by merge:
	Library/Formula/subversion.rb
Please, commit your changes or stash them before you can merge.
Aborting
Error: Failure while executing: git pull -q origin refs/heads/master:refs/remotes/origin/master
```

このエラーはgitでコンフリクトしているのでなんとかしろ、というエラーです。 なので、以下のコマンドで一旦最新ファイルで上書きします。

```sh
treemacpro:Formula tree$ cd /usr/local/Library/Formula
treemacpro:Formula tree$ git reset --hard HEAD
HEAD is now at 8acc7e6 postgis 2.1.0
```

続いてXCodeのコマンドラインツールをインストールするわけですが、 ネットで調べ見ると、以下のコマンドでコマンドラインツールをインストールしろ、とありました。

```sh
treemacpro:project tree$ xcode-select --install
```

しかしインストールの終盤に、このバージョンには対応してない云々と出てインストール失敗。

代わりに以下から手動でダウンロードできました。 [Sign in with your Apple ID - Apple Developer](https://developer.apple.com/downloads/index.action)[!\[\]\(//b.hatena.ne.jp/entry/image/https://developer.apple.com/downloads/index.action\)](http://b.hatena.ne.jp/entry/https://developer.apple.com/downloads/index.action) Command Line Tools (OS X Mavericks) for Xcode - Late October をクリックすると dmgファイルがDLできるので、インストールします。

さて、これでbrew updateしてみます。

```sh
treemacpro:Formula tree$ brew update
Updated Homebrew from 8acc7e6e to a59e2adf.
==> New Formulae
・・・略・・・
```

よし、成功！！

続いてbrew upgradeしてみます。

```sh
treemacpro:Formula tree$ brew upgrade
==> Upgrading 9 outdated packages, with result:
freetype 2.5.0.1, imagemagick 6.8.7-0, mercurial 2.7.2, multimarkdown 4.3.1, neon 0.30.0, nginx 1.4.3, ninja 1.4.0, serf 1.3.2, sqlite 3.8.0.2
==> Upgrading freetype
・・・略・・・
```
