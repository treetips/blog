[XML](http://d.hatena.ne.jp/keyword/XML)を整形したい時、ツールを使う方が多いと思いますが、macなら別途ツールを用意する必要ありません。 コマンド１発です。

以下の内容で[シェルスクリプト](http://d.hatena.ne.jp/keyword/%A5%B7%A5%A7%A5%EB%A5%B9%A5%AF%A5%EA%A5%D7%A5%C8)を作り、予め[クリップボード](http://d.hatena.ne.jp/keyword/%A5%AF%A5%EA%A5%C3%A5%D7%A5%DC%A1%BC%A5%C9)に[XML](http://d.hatena.ne.jp/keyword/XML)をコピーしておき、[シェルスクリプト](http://d.hatena.ne.jp/keyword/%A5%B7%A5%A7%A5%EB%A5%B9%A5%AF%A5%EA%A5%D7%A5%C8)を実行すると、 整形済み[XML](http://d.hatena.ne.jp/keyword/XML)が標準出力されます。

```
#!/bin/sh
pbpaste | xmllint --format -
```

標準出力ではなくクリップボードにコピーしたい場合は以下のようにリダイレクトする。

```
#!/bin/sh
pbpaste | xmllint --format - | pbcopy
```

xmllintって凄い便利ですね。 ちょっと試してみたんですが、これ、xmllintが入ってればLinuxでも動きます。 さくらのVPSはCentOSなんですが、標準でインストールされてました。

```
$ echo "<xml>test</xml>" | xmllint --format -
<?xml version="1.0"?>
<xml>test</xml>
```
