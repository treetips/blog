[Home &mdash; Playframework](http://playdocja.appspot.com/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://playdocja.appspot.com/\)](http://b.hatena.ne.jp/entry/http://playdocja.appspot.com/) 最近触りだしたplayです。 [google guava](http://code.google.com/p/guava-libraries/)[!\[\]\(http://b.hatena.ne.jp/entry/image/http://code.google.com/p/guava-libraries/\)](http://b.hatena.ne.jp/entry/http://code.google.com/p/guava-libraries/) そしてこの[google](http://d.hatena.ne.jp/keyword/google)製[java](http://d.hatena.ne.jp/keyword/java)ライブラリである、guavaを組み込もうとしたところ、少しはまりました。 guavaですが、今まで[apache](http://d.hatena.ne.jp/keyword/apache)のcommons-lang、io、collections.等を使っていましたが それを更に便利に今風にしたもので、おすすめなのです。

dependenciesにguavaを追加する まず、$PROJECT_HOME/project/Build.[scala](http://d.hatena.ne.jp/keyword/scala) を開きます。 val appDependencies = Seq( の部分を以下のように修正します。

```yaml
val appDependencies = Seq(
      // Add your project dependencies here,
      "mysql" % "mysql-connector-java" % "5.1.20"
      ,"com.google.guava" % "guava" % "13.0"
    )
```
