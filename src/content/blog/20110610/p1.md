nginxで[グローバルIP](http://d.hatena.ne.jp/keyword/%A5%B0%A5%ED%A1%BC%A5%D0%A5%EBIP)でIP制限を行ってみます。 [apache](http://d.hatena.ne.jp/keyword/apache)と大差はありません。

```conf
location /hoge/ {
    allow xxx.xxx.xxx.xxx;
    deny all;
    proxy_pass http://localhost:8080/fuga/;
}
```

この例では、 /hoge/にアクセスすると、IPが xxx.xxx.xxx.xxx の場合のみ表示する事ができます。

以下の場合は、特定のIPを許可し、他は全て拒否する、という意味になります。 allow xxx.xxx.xxx.xxx deny all

以下の場合は、全てを許可するが、特定のIPは拒否する、という意味になります。 allow all deny xxx.xxx.xxx.xxx
