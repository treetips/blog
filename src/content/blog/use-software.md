私が開発やネット関連で実際に使用している有償・無償のMACのソフトウェアを紹介します。

![f:id:treeapps:20170829002500p:plain](/hatena-images/images/fotolife/t/treeapps/20170829/20170829002500.png)

### OS周辺

#### XtraFinder

![f:id:treeapps:20150104155256p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104155256.png) [XtraFinder adds Tabs and features to Mac Finder.](https://www.trankynam.com/xtrafinder/)

OS標準のfinderを拡張したソフトウェアです。タブ機能やカラムの幅自動調節機能等、標準ではできない事が沢山できるようになります。

#### Mail.app

![f:id:treeapps:20150104155810p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104155810.png) Mac標準のメーラーです。今のところ標準メーラーで満足しています。会社では非力でディスク容量が少ないなmacbook airなので、web版gmailを使っています。

#### Dropbox

![f:id:treeapps:20150104160103p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104160103.png) [Login - Dropbox](https://www.dropbox.com/home)

ファイル共有ソフトです。Googleドライブやskydrive等もありますが、昔から使っているDropboxを愛用しています。

### WEB

#### 1Password

https://itunes.apple.com/jp/app/1password/id443987910?mt=12&uo=4&at=1l3v6pb パスワードの管理ソフトです。もうパスワードを誕生日や電話番号や住所の一部を使うのはやめましょう。この1Passwordを使えば、パスワードジェネレータによってランダムな50桁のパスワードを簡単に生成でき、それを記憶させる事ができます。一度記憶させれば、ボタン1発で対象サイトのログイン画面に遷移＋ログインボタンクリックを自動で行ってくれます。主要ブラウザのプラグインが用意されていて簡単にログインが必要なサイトにアクセス＋ログインする事ができます。

Mac Appの中で最高峰のソフトだと思います。これを使わない人は正直相当損してると思います。数ヶ月おきにセール（50%オフもありました）しているので、セールのタイミングを見計らって購入する事をお勧めします。

### マルチメディア

#### Audirvana Plus

![f:id:treeapps:20150104160432p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104160432.png) [Audirvana Plus &ndash; The Sound of your Dreams](http://audirvana.com)

音楽プレーヤーです。恐らくmacで最も音質がよく、最新音楽フォーマット(DSD等)にも対応しており、iTunesと連携し、プレイリスト部分だけをiTunes、再生はaudirvana、という連携が可能です。

#### VLC

![f:id:treeapps:20150104160848p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104160848.png) [Official download of VLC media player, the best Open Source player - VideoLAN](http://www.videolan.org/vlc/)

動画プレーヤーです。デフォルトで大量のコーデックが入っており、mac最強？の動画再生プレーヤーです。グラフィックボードによる再生支援も可能なので、かなり低負荷で動画を再生できます。

#### Tweetbot

twitterクライアントです。macのtwitterクライアントでかなりいい方だと思います。ただ、ウインドウ幅の最低幅が固定されているので、一定以下の幅に変更できない点が難。あと日本語ではなく英語のUIなので英語が苦手な方は注意です。

#### cooViewer

[cooViewer](http://www.geocities.jp/coo_ona/viewer.html)

画像ビューアです。zipに圧縮された画像も参照可能で、矢印キーで次の画像・前の画像等が簡単に移動できます。

### 開発

#### eclipse

![f:id:treeapps:20150104161737p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104161737.png) [Eclipse - The Eclipse Foundation open source community website.](https://eclipse.org)

プログラム開発統合ツールです。様々なプログラム言語を開発可能で、特にjavaの開発では世間一般ではeclipseが使用されます。私も開発でずっとeclipseを使っています。eclipseはSWTで開発されているため少し重いです。

#### IntelliJ IDEA

![f:id:treeapps:20150104164512p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104164512.png) [IntelliJ IDEA: The Java IDE for Professional Developers by JetBrains](https://www.jetbrains.com/idea/)

プログラム開発統合ツールです。eclipseより一歩も二歩も進んだ機能を持ち、eclipseより軽い開発ツールです。無料版もありますが、機能が限定されています。

#### Sublime Text

[Sublime Text - A sophisticated text editor for code, markup and prose](http://www.sublimetext.com) ネイティブアプリなので、非常に高速でリソース消費が少なく、高機能なエディタです。開発速度はatomより遅いですが、その分ネイティブアプリの高速性を発揮してくれます。私は仕事ではatomを使わずSublime Textオンリーです。

#### atom

![f:id:treeapps:20150104162349p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104162349.png) [Atom](https://atom.io)

Github社が開発しているテキストエディタです。開発は非常に活発で、パッケージは非常に多く、物凄く沢山の機能を追加する事ができます。しかしV8エンジンベースのnode.jsアプリであり、ネイティブアプリではありません。ネイティブではないのでパフォーマンス面ではSublime Textの10〜100倍遅い・重い場合があります。

#### iTerm2

![f:id:treeapps:20150104162704p:plain](/hatena-images/images/fotolife/t/treeapps/20150104/20150104162704.png) [iTerm2 - macOS Terminal Replacement](http://iterm2.com)

macのターミナルソフトです。タブに対応し、ウインドウを縦横に分割可能で、複数サーバのログ監視等がし易いmacの開発ツールの必需品です。
